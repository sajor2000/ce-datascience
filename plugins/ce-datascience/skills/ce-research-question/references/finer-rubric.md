# FINER Scoring Rubric

Five dimensions, scored 1-5 each, with a written rationale. Low scores are diagnostic, not disqualifying.

## F — Feasible (1-5)

- **5** — Data already in hand; cohort size confirmed adequate by `/ce-power`; team has methods experience; timeline < 12 months
- **4** — Data accessible with effort (DUA in hand or pending); team has 80% of methods experience needed
- **3** — Data accessible in principle (would require IRB amendment, vendor renewal, or new DUA); methods are stretch
- **2** — Data exists somewhere but access is not yet established; or team has no experience with the required methods
- **1** — Data does not exist in usable form, or required methods are not in the team's repertoire and a learning curve would dominate the timeline

Rationale must name the data source, the cohort-size estimate, and the methods bottleneck.

## I — Interesting (1-5)

- **5** — Active controversy in current journals or clinical guidelines; would change practice or alter a guideline if answered
- **4** — Clinically important question; field is debating it; answer would inform a smaller subgroup of practice
- **3** — Useful contribution; field is broadly aware of the question
- **2** — Niche audience; answer would interest a small subset of researchers
- **1** — Question is settled or has been answered repeatedly with consistent results

Rationale must cite at least one current paper, guideline, or controversy.

## N — Novel (1-5)

- **5** — Not previously studied in this population, this exposure, or this outcome
- **4** — Studied once or twice; current evidence has clear methodological limitations this study addresses
- **3** — Replication in a new context (different country, different healthcare system, different decade) — replication is valuable, score it as 3 not 1
- **2** — Mostly answered; this study would add precision but not direction
- **1** — Already answered with high-quality evidence; redoing it does not advance the field

Rationale must cite the closest prior paper. If you cannot find one, the novelty score is suspect — re-search the literature before claiming 5.

## E — Ethical (1-5)

- **5** — Observational with HIPAA waiver, IRB-exempt, or IRB-expedited
- **4** — Observational with full IRB but standard-of-care comparator; no consent burden beyond a HIPAA-style waiver
- **3** — Interventional with low-risk intervention; consent obtainable; clinical equipoise documented
- **2** — Interventional with non-trivial risk; consent challenging; equipoise contested
- **1** — Equipoise absent; intervention has known harm in a population that cannot meaningfully consent

Rationale must explicitly address equipoise (for interventional) or consent / re-identification risk (for observational).

## R — Relevant (1-5)

- **5** — Effect size large enough that a positive answer would change practice; effect size small enough that a null result would settle the question
- **4** — Plausible effect size that would change practice in a meaningful subgroup
- **3** — Useful for the field; not directly practice-changing
- **2** — Adds knowledge but no clear path from result to action
- **1** — No realistic decision the answer would change

Rationale should name the practice change a positive (or null) result would trigger.

## How `/ce-research-question` uses scores

- All scores ≥ 3: proceed without comment
- Any score = 2: surface the issue and ask the user whether it is acknowledged or whether the question should be reframed
- Any score = 1: stop and tell the user the question has a serious gap; offer to refine the question rather than proceed

A 1 on N (already answered) is not a refusal — replication studies are valuable when explicitly framed as replication. The skill should ask: "Are you framing this as a replication study? If yes, the suggested reporting checklist becomes the original study's checklist plus a 'replication' framing in SAP-1."
